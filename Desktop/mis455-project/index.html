Welcome to My MIS 455 Site

Assignment 1 — Five-Page HTML & CSS Project

What Is This Site?

This website was created as part of MIS 455, Assignment 1. The goal is to demonstrate basic web development skills using HTML5 and CSS3. Each page covers a different topic and is linked together through a shared navigation bar. The project is hosted on Netlify and version-controlled via GitHub.

Throughout this project you will find examples of flexbox layout, CSS grid, Google Fonts, image embedding, and a consistent stylesheet applied across all five pages. It is intentionally kept simple to reflect a starter-level project structure.

Getting Started With Web Dev

HTML is the skeleton of every webpage — it gives structure through elements like headings, paragraphs, images, and links. CSS adds the visual layer on top: colors, spacing, fonts, and layouts.

Together they form the foundation of front-end web development. Mastering these two technologies is the essential first step whether you are building a portfolio, a business page, or a class assignment.
